# Suma de Arreglos en Paralelo con OpenMP

Este proyecto implementa la suma de dos arreglos utilizando programación paralela con OpenMP en C++.

## Descripción
Se crean dos arreglos A y B con valores generados mediante una fórmula y un tercer arreglo C que almacena la suma A + B.

La suma se realiza en paralelo utilizando  `#pragma omp for`

## Herramientas utilizadas
- C++
- OpenMP
- Visual Studio
